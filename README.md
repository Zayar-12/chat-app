# 💬 ZayChat - Real-Time Chat Application

[![Laravel Version](https://img.shields.io/badge/Laravel-12.x-red.svg)](https://laravel.com)
[![Livewire Version](https://img.shields.io/badge/Livewire-3.x-pink.svg)](https://livewire.laravel.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

ZayChat is a modern, real-time chat application built using **Laravel 12**, **Livewire**, and **Tailwind CSS**. It provides a seamless, dynamic messaging experience without the need for full page refreshes.

---

## 🚀 Features

- **Real-Time Messaging**: Instant message delivery and updates powered by Livewire reactive states.
- **Live User Search**: Quickly filter and find users instantly as you type.
- **New Chat Modal**: A clean, elegant modal interface to seamlessly start new conversations.
- **Robust Authentication**: Secure system integrated with Google OAuth Login and standard Email Verification.
- **Component-Based UI**: Highly refactored, modular Blade components for optimal code maintainability and speed.
